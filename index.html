Atalhos Vscode

(voltar)

São exatamente esses marcadores! Existem inúmeros marcadores/tags para deixar o seu conteúdo o mais estruturado e organizado possível. O HTML5 nos ajudou nessa missão, trazendo muitas tags novas!

Temos tags para botões, formulários, imagens, tabelas, textos, entre outras.

Tags

(voltar)

usar o b - deixa tudo em negrito

usar o i - deixa tudo itálico

usar o u - deixa tudo sublinhado

usar o sup - deixa tudo acima da linha

Atributos especificos

(voltar)

usar o strong+ID - identifica o elemento que sofre alteração

usar o strong+style - identifica uma cor a escolher

usar o strong+class - coloca o ponto principal com determinada cor

usar o blockquote - isso coloca o texto como citação

usar o h no inicio da linha determina o titulo e seus subtitulos pela numeracao apos o h1,h2,h3...

usar o p no inicio da linha determina que inicio o corpo do texto.
